The Soundgate Core Mobile Audio Solution

Introducing the Core from SoundGate. Core will allow you to pass audio and video from your Portable Media Device. Core is completely universal; all you need is an auxiliary input on your radio. Core operates by simply connecting power via the supplied cigarette adaptor plug and using the provided AV cables to connect to your existing auxiliary input. Core is also a charging source, enabling unlimited use while driving. SoundGate's Core is the first product to provide in-vehicle charging for the Zune, as well as the first universal integration solution for all your portable audio and video needs.
