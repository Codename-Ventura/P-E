HOW TO USE THIS CODE GENERATOR:
1.) Enter the item title that you wish displayed in the ITEM TITLE box and press ADD TITLE. Be sure to write this title down because you will have to re-enter it in the eBay Item Title Box when you list your item.
2.) Enter a single descriptive bullet point about the item in the ITEM DESCRIPTION BULLET POINTS box (Ex. This item is Black) and then press ADD INFO.
3.) Continue to add more bullet points, pressing ADD INFO in between each one, until you have accurately described the item.
4.) Select the proper Shipping method (Shipping fees are included OR are not included in the final price) and then press ADD INFO.
5.) Enter any additional info that concerns the auction but does not further describe the item (EX: We do not accept Personal Checks) in the ADDITIONAL INFO BULLET POINTS section. Be sure to press ADD INFO after each entry.
6.) Once you have worked you way down from the top of the page, staying in order and pressing ADD INFO after each entry, then click on the CREATE HTML button. This will generate the html code required for the eBay Item Listing..
7.) Once the html code is generated, you can press on the VIEW HTML PAGE button to preview your eBay listing. Click your BACK button to go back to the html code view. If there are any errors on the preview page, you must go back to the html code view and DELETE ALL OF THE CODE AND START OVER COMPLETELY. You can delete the generated code by clicking BOTH of the CLEAR ALL HTML ABOVE buttons.
8.) If you are happy with the code you have generated, press the SELECT CODE button to select the code. Once the code is selected, you will need to copy the code (Control-C) and then paste it in your eBay Item Listing Description section. Be sure to select that you are entering HTML code in your eBay Item Listing Description.
9.) If you are done with the code and wish to create a new eBay Item Listing, simply press BOTH of the CLEAR ALL HTML ABOVE buttons to start a new listing.
